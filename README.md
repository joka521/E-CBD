# E-CBD - Plateforme de Vente de Fleurs de CBD Premium

## Description

E-CBD est un site e-commerce spécialisé dans la vente de fleurs de CBD légales et premium en France. Le site propose une sélection des meilleures variétés provenant de **L'Entrepôt du Chanvrier**, un grossiste réputé spécialisé dans le chanvre légal.

## Structure du Projet

```
E-CBD/
├── index.html                 # Page d'accueil
├── shop.html                  # Page de la boutique
├── contact.html               # Page de contact
├── product-*.html             # Pages produits individuels
├── assets/
│   ├── css/
│   │   └── style.css           # Styles responsive modernes
│   └── img/                  # Images des produits
└── legal/
    ├── mentions-legales.html
    ├── cgv.html
    └── politique-confidentialite.html
```

## Images des Produits

Les images des produits sont actuellement des images de placeholder (Unsplash). Pour la version de production, remplacer par les images réelles du grossiste L'Entrepôt du Chanvrier.

### Structure des URLs d'images

```html
<!-- Actuel (placeholder) -->
<img src="https://images.unsplash.com/photo-xxx?auto=format&fit=crop&w=500&q=80" alt="Nom produit">

<!-- Production (à remplacer avec URLs réelles) -->
<img src="assets/img/produit-nom.jpg" alt="Nom produit">
```
